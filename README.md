
<h1>Notificacion de cortes por mail</h1>

<p>Herramienta que permite automatizar el proceso de notificación de productos faltantes a sucursales por medio de correo electrónico.</p>

<p>Implementación en Python que funciona como un ejecutable, utiliza las librerías de Pandas y NumPy para la manipulación de DataFrames, la librería de openpyxl y la de Tkinter para la interfaz del usuario y la de smtplib para el envío de correos electrónicos por medio del protocolo SMTP. </p>

<p>Características: </p>
<ul><li>Es portable y se puede ejecutar en otra PC</li></ul>
<ul><li>Tiene un modo interactivo: El usuario puede seleccionar de la aplicación un botón para cargar archivos desde su PC</li></ul>
<ul><li>Para su procesamiento de datos, se hace uso de archivos con extensión xlsx. Las columnas deben de manetener un orden específico </li></ul>
